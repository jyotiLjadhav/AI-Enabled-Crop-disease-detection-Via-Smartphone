About This Project

Project Overview

This project is a Samsung Innovation Campus Capstone project focused on using AI and Deep Learning for smart agriculture.

The system uses a Convolutional Neural Network (CNN) trained on the Kaggle Plant Disease dataset to detect diseases from smartphone leaf images.

Key features include Grad-CAM explainability, prediction history, and a user-friendly web interface designed for farmers.
